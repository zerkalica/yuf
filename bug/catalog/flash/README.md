# $yuf_bug_catalog_flash

Randomly flasning menu cards if F5 and scroll

open locally `http://localhost:9080/yuf/bug/catalog/flash/-/test.html#!ids_get_timeout=100/spread_ids_refresh_rate=1000/ids_max=50`

Or [github link](https://zerkalica.github.io/yuf/#!demo=yuf_bug_catalog_flash/ids_get_timeout=100/spread_ids_refresh_rate=1000/ids_max=50)

## Url params

* `ids_get_timeout` - number, ids get async timeout emulation, 0 - disables async get
* `spread_ids_refresh_rate` - number, ms, list refresh rate, 0 - disables refresh
* `ids_max` - max cards count in list
